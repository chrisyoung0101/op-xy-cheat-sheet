OP-XY Marker Breakdown

<><> Button Combos

  • Hold Step to check/edit notes or add chord to step
  • Hold Step and press another step to extend the Note
  • Hold Step and press PLUS or MINUS to nudge the note (quantization off)
  • Hold key and press Record to show notes or press Step to record
  • Hold a step and adjust an encoder to lock a parameter
  • What number step is that? HOLD the step and the number will show
  • Record + Stop deletes current track
  • Select a bar: PRESS bar to advance
  • See what is on a step: hold it down, assigned keys light
  • Add step component: SHIFT + step, step flashes
  • Access instrument menu: Instrument ACTIVE, HOLD shift + Instrument
  • Granular tempo: SHIFT + adjust tempo
  • Link tracks: hold source track, press target track
  • Preset Options: hold track > M1-M4 for scramble, copy, save
  • Save preset: hold track > M4 save
  • SHIFT + Sampler: load sample
  • Mixer access: Mixer + M2 for EQ, Mixer twice for Aux mode
  • COM Mode exit: SHIFT + COM

<><> tip (Information / Advice)

  • Pad not playing right? Remove velocity in instrument menu
  • Each track has independent bar measures
  • Punch-in FX: upper octave for melodic, lower for percussion
  • The last three buttons are G-force sensitive
  • Pulse step component: replays step, causes desync unless fixed
  • Pulse + Jump = stay in sync
  • Skip Trigger/Step/Param Lock Components: think "triggering" not "skipping"
  • Arrange Mode: scenes show mute/mix info
  • Arrange beyond 9 scenes: SHIFT + 0 + two-digit number
  • FX I to FX II: stack effects
  • Master Saturator is distortion
  • Presets exportable
  • Axis = FM synth good for lush strings
  • Organ type param 1 = Transistor / 8 = Church
  • COM Mode has useful system settings
  • MTP: sends files via connected computer
  • Firmware updates via Web MIDI or boot mode

<><> encoder (Encoders & Their Function)

  • Bar Menu: Quantization, Length (Gate), Groove, Shape
  • Mixer Mode: FXI, FXII, Pan, Level
  • Master EQ: Low, Mid, High, All
  • Players Arpeggio: Rate, Pattern, Octaves / SHIFT: Note Length, Variation, Gliss, Stereo
  • Players Maestro: Roll, Variation, (N/A), Hold
  • Axis: Tone, Ratio, Shape, Tremolo
  • Dissolve: Swarm, AM, FM, Detune
  • EPiano: Tone, Texture, Punch, Time
  • Hardsync: Freq, Sub, Noise, Lowcut
  • Organ: Organ type, Sub Bass, Tremolo Amount, Rate
  • Prism: Shape, Ratio, Detune, Stereo
  • Simple: Shape, Pulse Width, Noise, Stereo
  • Wavetable: Table, Position, Warp, Drift

  • Instrument Play Mode: M2 + SHIFT

<><> tape modules M1-M4 (Tape-Specific Controls)

  • M1: Tape UI
  • M2: Audio Send
  • M3: Filter
  • M4: LFO

<><> fx params (FX Parameters)

  • Chorus: Rate / Depth / Feedback / Stereo

  • Distortion: Drive / Clip / Low-cut / Hi-cut
  • Lofi: Rate / Bits / Quality / Drift
  • Phaser: Frequency / Depth / Rate / Feedback
  • Reverb: Size / Mod / Tone / Dry-to-Wet
